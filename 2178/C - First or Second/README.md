Santa is deciding which children belong on the nice list and which belong on the naughty list.</p><p>An integer $$$X$$$ is initially set to $$$0$$$. Santa will perform the following operation <span class="tex-font-style-bf">exactly</span> $$$n-1$$$ times:</p><ul> <li> Choose the first or second child in line and remove them from the line. </li><li> Let $$$w$$$ be the <span class="tex-font-style-it">niceness</span> of the chosen child. <ul> <li> If the first child was chosen, add them to the nice list and add $$$w$$$ to $$$X$$$. </li><li> If the second child was chosen, add them to the naughty list and subtract $$$w$$$ from $$$X$$$. </li></ul> </li></ul><p>Note that after all operations, exactly one child remains unassigned to a list.</p><p>Determine the maximum possible value of $$$X$$$ that Santa can obtain after all $$$n-1$$$ operations.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. contains $$$n$$$ integers $$$a_1,a_2,\ldots,a_n$$$ ($$$-10^9\le a_i\le 10^9$$$) — the <span class="tex-font-style-it">niceness</span> of each child.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$2\cdot 10 ^ 5$$$. </p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, print a single integer — the maximum
