<h3><a href="https://codeforces.com/contest/2178/problem/G" target="_blank" rel="noopener noreferrer">deCH OR Dations</a></h3>

<div class="header"><div class="title">G. deCH OR Dations</div><div class="time-limit"><div class="property-title">time limit per test</div>3 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p> </p><p>Due to supply chain issues in the North Pole (allegedly caused by a lazy elf), Santa is planning to give out hand-drawn circles as presents this Christmas. Please help him decorate them!</p><p>There are $$$2n$$$ equally spaced points around the circle, labeled $$$1,2,\ldots,2n$$$ in clockwise order. Santa has chosen $$$n$$$ chords with <span class="tex-font-style-bf">distinct</span> endpoints, where chord $$$i$$$ connects the points $$$a_i$$$ and $$$b_i$$$. He draws these chords onto the circle one by one, in order. </p><p>After Santa has drawn the first $$$\ell$$$ chords, consider any non-empty subset $$$S$$$ of the $$$\ell$$$ chords and let $$$1\leq c_1<c_2<\cdots<c_{|S|}\leq \ell$$$ be their indices. $$$S$$$ is defined to be a <span class="tex-font-style-it">chain</span> if and only if chord $$$c_k$$$ intersects chord $$$c_{k+1}$$$ for all $$$1\leq k < |S|$$$. Note that $$$S$$$ is always a <span class="tex-font-style-it">chain</span> if $$$|S|=1$$$.</p><p>Santa does not want any chord to be the odd one out. Therefore, he considers the chords to be <span class="tex-font-style-it">tight-knit</span> if and only if every chord appears in an even number of <span class="tex-font-style-it">chains</span>, taken over all <span class="tex-font-style-it">chains</span> that are subsets of the first $$$\ell$$$ chords.</p><p>For each $$$\ell$$$ from $$$1$$$ to $$$n$$$, help Santa determine whether his chords are <span class="tex-font-style-it">tight-knit</span>.</p></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains a single integer $$$n$$$ ($$$2\le n\le 5\cdot 10^5$$$) — the number of chords.</p><p>Then $$$n$$$ lines follow, the $$$i$$$-th line containing two integers $$$a_i$$$ and $$$b_i$$$ ($$$1\le a_i<b_i\le 2n$$$) — the endpoints of the $$$i$$$-th chord.</p><p>It is guaranteed that all endpoints are distinct.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$5\cdot 10^5$$$. </p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a string of length $$$n$$$ — the $$$\ell$$$-th character should be $$$\texttt{1}$$$ if the first $$$\ell$$$ chords are <span
