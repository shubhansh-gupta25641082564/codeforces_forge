<h3><a href="https://codeforces.com/contest/2178/problem/F" target="_blank" rel="noopener noreferrer">Conquer or of Forest</a></h3>

<div class="header"><div class="title">F. Conquer or of Forest</div><div class="time-limit"><div class="property-title">time limit per test</div>2 seconds</div><div class="memory-limit"><div class="property-title">memory limit per test</div>512 megabytes</div><div class="input-file input-standard"><div class="property-title">input</div>standard input</div><div class="output-file output-standard"><div class="property-title">output</div>standard output</div></div><div><p>   </p><p>Define the unique <span class="tex-font-style-it">ornamental</span> coloring of a rooted tree$$$^{\text{∗}}$$$ as the following vertex coloring: </p><ul> <li> A vertex $$$v$$$ is colored white if the number of vertices in the subtree$$$^{\text{†}}$$$ rooted at $$$v$$$ is even; </li><li> Otherwise, $$$v$$$ is colored black. </li></ul><p>On his quest to conquer a forest of Christmas trees, Yuuki encountered an <span class="tex-font-style-it">ornamentally</span> colored tree $$$T$$$ with $$$n$$$ vertices labeled from $$$1$$$ to $$$n$$$, rooted at vertex $$$1$$$.</p><p>Yuuki considers the tree <span class="tex-font-style-it">conquered</span> if and only if <span class="tex-font-style-bf">at least one</span> of the following conditions holds:</p><ul> <li> There are no white vertices in the tree, or </li><li> There exists some vertex $$$v$$$ such that all white vertices lie on the simple path from the root $$$1$$$ to $$$v$$$. </li></ul><p>To <span class="tex-font-style-it">conquer</span> the tree, Yuuki can apply the following operation on $$$T$$$ an arbitrary number of times (possibly zero): </p><ul> <li> First, choose a vertex $$$w$$$ that is colored white and is <span class="tex-font-style-bf">not</span> the root of $$$T$$$. Let $$$p_w$$$ be the parent of $$$w$$$. </li><li> Then, remove the edge connecting $$$p_w$$$ and $$$w$$$, and add an edge between <span class="tex-font-style-bf">any</span> two vertices such that $$$T$$$ remains a tree. </li><li> Finally, recolor the vertices of $$$T$$$ such that it is <span class="tex-font-style-it">ornamentally</span> colored. Note that $$$T$$$ is always rooted at vertex $$$1$$$. </li></ul><center> <img class="tex-graphics" src="https://espresso.codeforces.com/6531fb7f6eb29b4dc2070d58cdb89213095b5957.png" style="max-width: 100.0%;max-height: 100.0%;" width="756px">  <span class="tex-font-size-small">A possible application of the operation in the first test case. The resulting tree is <span class="tex-font-style-it">conquered</span> since all white vertices lie on the path between vertices $$$1$$$ and $$$3$$$.</span> </center><p>Compute the number of distinct$$$^{\text{‡}}$$$ <span class="tex-font-style-it">conquered</span> trees that Yuuki can construct by applying the above operation an arbitrary number of times on $$$T$$$. Since the answer may be large, output it modulo $$$998\,244\,353$$$.</p><p>Note that Yuuki cannot stop midway through an operation (in particular, he must recolor the tree before checking if it is <span class="tex-font-style-it">conquered</span>). Additionally, Yuuki is allowed to apply the operation even if the tree is already <span class="tex-font-style-it">conquered</span>.</p><div class="statement-footnote"><p>$$$^{\text{∗}}$$$A tree is a connected graph without cycles. </p><p>$$$^{\text{†}}$$$A subtree of vertex $$$v$$$ is the subgraph of $$$v$$$, all its descendants, and all the edges between them. </p><p>$$$^{\text{‡}}$$$Two trees are considered distinct if and only if there exists a pair of vertices such that there is an edge between them in one of the trees, and not in the other.</p></div></div><div class="input-specification"><div class="section-title">Input</div><p>Each test contains multiple test cases. The first line contains the number of test cases $$$t$$$ ($$$1 \le t \le 10^4$$$). The description of the test cases follows. </p><p>The first line of each test case contains a single integer $$$n$$$ ($$$2\le n\le 2\cdot 10^5$$$) — the number of vertices in $$$T$$$.</p><p>Then $$$n-1$$$ lines follow, the $$$i$$$-th line containing two integers $$$u_i$$$ and $$$v_i$$$ ($$$1\le u_i<v_i\le n$$$) — the two vertices that the $$$i$$$-th edge connects.</p><p>It is guaranteed that the given edges form a tree.</p><p>It is guaranteed that the sum of $$$n$$$ over all test cases does not exceed $$$2\cdot 10^5$$$. </p></div><div class="output-specification"><div class="section-title">Output</div><p>For each test case, output a single integer — the number of distinct <span class="tex-font-style-it">conquered</span> trees that can be constructed from $$$T$$$, modulo $$$998\,244\,353$$$.</p></div><div class="sample-tests"><div class="section-title">Example</div><div class="sample-test"><div class="input"><div class="title">Input<div title="Copy"
