\ldots, a_{n - 1}, b_0, \ldots, b_{m - 1}$$$ is a permutation of integers from $$$1$$$ to $$$n + m$$$.</p><p>The sum of $$$n + m$$$ over all testcases doesn't exceed $$$2 \cdot 10^5$$$.</p></div><div
